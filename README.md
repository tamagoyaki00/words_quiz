## ◆世代別言葉クイズ
全5問ランダムに出題され、言葉の意味を当てるクイズゲームです。
昭和・平成・Z世代（おまけで２ちゃんねる）のカテゴリーに分かれています。

### ◆ユーザー層
最近の言葉わからない・・・。現代にちょっぴり遅れを感じている方
昔の言葉わからない・・・。昔の流行りを知りたい方
そういえば昔こういうワードが流行っていた！と懐かしみたい方

### ◆使い方
クイズのカテゴリを選ぶ
「始める」を押す
クイズに答える
★よかったらｘシェアしてください！
※全問正解者のみ違うページに辿り着くので、全問正解するまで挑戦してみてください。
